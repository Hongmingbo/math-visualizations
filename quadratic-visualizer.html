📚 知识点讲解

二次函数的一般形式：y = ax² + bx + c (a ≠ 0)

  • a 的作用：决定开口方向和大小
    • a > 0：开口向上，有最小值
    • a < 0：开口向下，有最大值
    • |a| 越大，开口越窄；|a| 越小，开口越宽
  • b 的作用：影响对称轴位置
    • 对称轴公式：x = -b/(2a)
  • c 的作用：决定与y轴交点
    • 当 x = 0 时，y = c
  • 顶点坐标：(-b/2a, (4ac-b²)/4a)
  • 判别式 Δ = b² - 4ac：
    • Δ > 0：两个不同实根
    • Δ = 0：一个实根（重根）
    • Δ < 0：无实根
