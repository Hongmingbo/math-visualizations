完全平方公式面积拼图

通过正方形面积的不同计算方式，可以直观地证明完全平方公式。

(a + b)² = a² + 2ab + b²
  • 整体看：大正方形的边长是 (a+b)，面积为 (a+b)²。
  • 拆分看：大正方形由四个部分组成：一个面积为 a² 的红正方形，一个面积为 b² 的蓝正方形，以及两个面积均为 ab 的黄矩形。

勾股定理（赵爽弦图）

这是中国古代数学家赵爽利用“割补法”证明勾股定理的经典弦图。

a² + b² = c²
  • 初始状态下，四个全等的直角三角形（边长 a,b,c）围成一个倾斜的大正方形，面积为 c²。
  • 移动三角形后，空白部分变成了两个小正方形，面积分别是 a² 和 b²。
  • 因为总面积和三角形面积不变，所以空白部分的面积一定相等。

配方法几何证明

将二次多项式转化为完全平方式的过程，可以通过几何拼图完美解释。

x² + bx + (b/2)² = (x + b/2)²
  1. 我们有一个面积为 x² 的大正方形和面积为 bx 的矩形。
  2. 将矩形平分，得到两个面积为 (b/2)x 的小矩形。
  3. 将它们分别拼在 x² 的右侧和下方。
  4. 此时图形缺了一个角，面积恰好是 (b/2)²，补上它就得到了完整的大正方形！

圆面积公式推导

通过极限思想“化曲为直”，将圆转化为我们熟悉的矩形。

S = πr²
  • 将圆等分成若干个小扇形。
  • 将这些扇形交错拼接，可以组成一个近似的平行四边形（或矩形）。
  • 当等分的份数无限多时，拼成的图形就是完美的矩形。
  • 矩形的长是圆周长的一半 (πr)，宽是半径 (r)。
