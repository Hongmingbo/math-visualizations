📚 知识点讲解

一次函数的一般形式：y = kx + b (k ≠ 0)

  • k - 斜率（倾斜程度）：
    • k > 0：直线从左下向右上倾斜，y 随 x 增大而增大
    • k < 0：直线从左上向右下倾斜，y 随 x 增大而减小
    • k = 0：水平直线（y = b）
    • |k| 越大，直线越陡峭
  • b - y轴截距：
    • 直线与 y 轴的交点坐标为 (0, b)
    • b > 0：交点在 y 轴正半轴
    • b < 0：交点在 y 轴负半轴
    • b = 0：直线过原点
  • x轴截距：
    • 令 y = 0，解得 x = -b/k
    • 直线与 x 轴的交点坐标为 (-b/k, 0)
  • 特殊直线：
    • y = x：第一、三象限角平分线，k = 1
    • y = -x：第二、四象限角平分线，k = -1
    • y = b：水平直线，k = 0
    • x = a：垂直直线（不是函数）
