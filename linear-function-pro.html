📚 核心知识点

🔴 斜率 k 的意义

  • k > 0：直线上升，y 随 x 增大而增大
  • k < 0：直线下降，y 随 x 增大而减小
  • k = 0：水平直线，y 恒定不变
  • |k| 越大，直线越陡峭

🟢 截距 b 的意义

  • 直线与 y 轴 的交点坐标为 (0, b)
  • b > 0：交点在 y 轴正半轴
  • b < 0：交点在 y 轴负半轴
  • b = 0：直线过原点

🔵 x 轴截距

  • 令 y = 0，解得 x = -b/k
  • 直线与 x 轴的交点坐标为 (-b/k, 0)
  • 当 k = 0 时，无 x 轴截距（水平线）

💡 特殊直线

  • y = x：一、三象限角平分线
  • y = -x：二、四象限角平分线
  • y = b：平行于 x 轴的水平线
  • x = a：平行于 y 轴的竖直线（不是函数）
